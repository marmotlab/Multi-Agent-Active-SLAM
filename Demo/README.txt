This folder contains 8 demos showcasing different planning strategies used in the Multi-Agent Active SLAM experiments:
- Random
- Random Explore
- Frontier Explore
- Max Cost Maps
- Fast Cost Maps
- Long-Term Cost Maps
- Fast Long-Term Cost Maps
- RRT (Rapidly-Exploring Random Tree)

For each demo, a GIF is provided to visualize what Agent 1 perceives, including its estimated uncertainty and how it calculates its plans. Additionally, more details about each episode are provided.

Enjoy exploring the different planners and their strategies!
